设计模式系列:组合模式

发表于 2017-11-09 | 分类于 java ， 设计模式 | |

概念

组合模式是把对象堆起来形成集合的一种方式。它组织对象形成集合，结合迭代器模式，可以对客户隐藏具体对象实现。不至于暴露集合内部信息。形式上经
常将组合你模式用于对象的树形结构表示。
